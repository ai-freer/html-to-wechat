把 Agent 生成的 HTML Artifact
投递到公众号、小红书、AI 视频

Claude / Codex 生成的 HTML 文章，排版和视觉远胜裸 Markdown，但公众号、小红书、视频软件各有自己的"语法"。 这里把"HTML 是源"的工作流跑通，针对每种发布形态做一套转换管线。

01

HTML → 公众号富文本

v0.1 可用

粘贴或上传 HTML，浏览器内做 CSS 内联、清理脚本、复制为富文本，直接粘到公众号后台编辑器。

无需安装 · 浏览器内运行 · 不上传服务器

→
02

HTML → 图文号 / 小红书图集

规划中

把长文章改造成"上图下文"的卡片图集（小红书、公众号图文号）。文字卡片为主、AI 插画为辅。

需要 Skill 驱动 · 自带图也可独立用页面预览

→
