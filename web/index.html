把 Agent 生成的 HTML Artifact 投递到 4 种发布形态

Claude / Codex 生成的 HTML 文章，排版和视觉远胜裸 Markdown，但公众号、小红书、视频软件、飞书各有自己的"语法"。 这里把"HTML 是源"的工作流跑通，针对每种发布形态做一套转换管线。

01 浏览器即用

HTML → 公众号富文本

粘贴 / 输入 URL / 上传 HTML，浏览器内 CSS 内联、清理脚本、复制为富文本，直接粘到公众号后台编辑器。

无需安装 · 浏览器内 · 不上传服务器

开始转换 →
02 样例可看

HTML → 小红书图集

把长文章自动转成 8-10 张统一风格的卡片图集 + 配套文案。所有图由顶级文生图模型生成。

看示例图集 →
03 浏览器即用

HTML → AI 视频口播稿

改写成 AI 视频生成软件（Sora / 即梦 / 海螺 / Pika）可直接喂的口播稿。短句、口语化、按节奏切段。

marked 预览 · 字数 / 时长 / 段落统计 · 下载 .md / .txt

开始转换 →
04 Skill 可用

HTML → 飞书云文档

HTML 直转 DocxXML → lark-cli docs +create，飞书云文档原生 Block 输出。支持 grid / callout / 画板（Mermaid 可拖）/ 远程图自动入库。

需要 @larksuite/cli · agent 模式 · 文档作者归属用户

看设计文档 ↗

每个模式都对应 docs/0X-...md 里的设计文档 · 跨模式范式 X-C / X-A 见 ROADMAP
