V0.5b · <span style="color: ..."> 颜色映射

本文档测试 inline CSS color / background-color → DocxXML span text-color / background-color 的映射。

1. 命名色（DocxXML 原生支持）

这段文字默认黑色，红色 red、蓝色 blue、绿色 green。

2. #hex 转 rgb()

三位 hex：#f00 → rgb(255, 0, 0)。六位 hex：#a4302a → 砖红、#1c5f3e → 墨绿。

3. rgb() / rgba() 兜底

rgb：蓝色 rgb 三元组。rgba：半透明 rgba（alpha 被丢）。

4. background-color

背景色：高亮黄底、桃色背景。

5. color + background-color 组合

同时：白字砖红底、黄字海军蓝底（navy 走 fallback null）。

6. 跳过的 case

这些不映射：var() 引用、hsl()、透明、继承。
